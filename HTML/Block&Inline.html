-Block Element-

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.A block-level element always takes up the full width available (stretches out to the left and right as far as it can).Two commonly used block elements are: p tag and div .

Examples :

  1. h1 tags
  2. p tag
  3. section tag
  4. div tag
  5. address tag
  6. aside tag
  7. li tag
  8. canvas tag
  9. footer tag
  10. main tag

-Inline Element-

An inline element does not start on a new line and it only takes up as much width as necessary.The div tag element is a block-level and is often used as a container for other HTML elements.The span tag element is an inline container used to mark up a part of a text, or a part of a document.

Examples :

  1. span tags
  2. img tag
  3. vedio tag
  4. audio Tag
  5. code tag
  6. button tag
  7. city tag
  8. a tag
  9. map tag
  10. object tag
